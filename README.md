## 🛠️Local development

```bash
# clone the repository
git clone git@github.com:themefisher/educenter-hugo.git

# setup project
$ npm run project-setup

# Start local dev server
$ npm run dev
```

Or Check out [Full Documentation](https://docs.gethugothemes.com/educenter/?ref=github).
